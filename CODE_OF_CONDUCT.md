# Code of Conduct

This is a professional workspace for builders. We maintain a high-signal, low-noise environment to maximize focus and output.

## Core Expectations

- **Respect:** Treat all contributors with professionalism. Constructive technical feedback is expected; personal attacks or harassment are strictly prohibited.
- **No Spam:** Keep discussions, issues, and pull requests strictly focused on technology, building, and relevant collaboration. Self-promotion, marketing, and off-topic content are not allowed.
- **Stay Relevant:** Ensure your contributions align with the project specifications and the broader technical goals of the network.

Violations of these principles will result in immediate removal from the repository and the broader network.
