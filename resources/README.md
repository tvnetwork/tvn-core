# Resources

A curated collection of technical documentation, system architectures, tools, and shared knowledge for TVN builders.

## Structure

Organize new resources as markdown files or subdirectories based on their technical domain (e.g., `architecture/`, `deployment/`, `tooling/`).

## Adding Resources

When contributing to this directory:
- Ensure the content is highly technical, accurate, and relevant to software development.
- Avoid generic or beginner tutorials; prefer linking to official documentation.
- Maintain a clean, consistent markdown format.
